Signal the GOLDi cloud, that the test for the given firmware file succeeded and that it is ready for deployment.
GetNextUntestedFirmwareAsync
Get a firmware file which was uploaded to the GOLDi cloud for automatic testing.
GetTestDeviceDefaultFirmwareAsync
Get a firmware file that will be applied to the test device when the test is completed.
