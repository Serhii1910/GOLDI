Remarks
Capturers can be used to efficiently notify a characteritic that a value changed. This can be done without any expensive string lookups. To use capturers this way, invoke the Bind(InsightIdentity) method.

The InsightCapturerExtensions class provides extension methods for this interface that simplify the use of the Capture(T, IEnumerableInsightOrigin) method at the cost of additional object allocations.

While the use of capturers can provide a high-performance interface to the logging system, there are more convinient extension methods on the InsightIdentity object provided by the InsightIdentityExtensions class.
